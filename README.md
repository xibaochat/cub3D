## Attention
This project was designed to work on a darwinOS, using the minilibX library.

## Usage

Launch the game :

> make

> ./Cub3D maps/my_map00.cub

Save a screenshot of the first image of the game :

> ./Cub3D maps/my_map00.cub --save
